Anything Search Support

Anything Search is a local-first file search utility for macOS. It indexes only folders you choose in the app.

Getting Started

  1. Open Anything Search.
  2. Open Settings, then Locations.
  3. Add a folder such as Downloads, Documents, or a project folder.
  4. Wait for indexing to finish.
  5. Search by file name, folder name, extension, or path fragment.

Troubleshooting

  • If a folder is missing, add it again from Settings > Locations so macOS can grant folder access.
  • If results look incomplete, wait for indexing to finish or reindex the selected location.
  • The first release searches file and folder names, paths, and basic metadata. It does not index file contents.

Privacy

File names, paths, search queries, and indexes stay on your Mac. Core search does not require a network connection.
